# Health Insurance Cost Prediction

This project predicts medical insurance charges using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Model Used
- Linear Regression

## Features
- Data preprocessing
- Visualization
- Model training and testing
- R2 score evaluation

## Output
Achieved ~0.74 R2 score on test data.
